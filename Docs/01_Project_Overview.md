# Project Overview

# BCI-Based Frequency-to-Color Encoding Framework for Future Cortical Color Restoration

---

# 1. Introduction

Vision is one of the most important human senses, allowing us to perceive the surrounding environment through light and colors. The human visual system converts incoming light into electrical signals, which are then processed by different regions of the brain to produce meaningful visual perception.

For individuals suffering from severe visual impairments or cortical vision disorders, the normal communication pathway between the eyes and the brain is disrupted, making it difficult or impossible to perceive visual information accurately. Although significant progress has been made in visual prosthetics and Brain-Computer Interface (BCI) technologies, most existing systems primarily focus on restoring basic visual perception such as brightness, shapes, or object detection. Very limited research has explored computational frameworks capable of representing color information for future cortical stimulation systems.

This project investigates a Brain-Computer Interface (BCI) framework that combines Electroencephalography (EEG), Steady-State Visual Evoked Potentials (SSVEP), signal processing, and computational neuroscience to develop a novel frequency-to-color encoding strategy. The long-term vision of this research is to contribute toward future cortical color restoration systems using non-invasive brain stimulation techniques.

---

# 2. Background

A Brain-Computer Interface (BCI) is a communication system that enables interaction between the human brain and external devices by measuring brain activity and converting it into useful information.

One of the safest and most widely used methods of recording brain activity is Electroencephalography (EEG). EEG records tiny electrical signals generated by neurons using electrodes placed on the scalp.

When a person continuously observes a flickering visual stimulus, neurons in the visual cortex begin oscillating at the same frequency as the stimulus. This neural response is known as the Steady-State Visual Evoked Potential (SSVEP).

Because SSVEP responses are reliable, require very little user training, and produce high signal-to-noise ratios, they have become one of the most popular paradigms for Brain-Computer Interface research.

---

# 3. Problem Statement

Current BCI systems successfully detect user intentions using EEG and SSVEP. However, the majority of existing research focuses on applications such as:

- Wheelchair control
- Spelling systems
- Cursor movement
- Robot control
- Smart home automation

Very little work has investigated computational frameworks capable of representing color information that may eventually be delivered back to the visual cortex.

Additionally, existing visual prosthetic systems primarily restore basic visual perception rather than detailed color perception.

Therefore, there is a need to investigate new computational methods that can encode decoded EEG information into meaningful color representations for future cortical stimulation systems.

---

# 4. Proposed Idea

This project proposes an EEG-based Brain-Computer Interface framework capable of detecting SSVEP frequencies and converting them into predefined color representations through a novel frequency-to-color encoding algorithm.

Unlike conventional SSVEP applications where detected frequencies directly represent commands (for example, selecting a letter or controlling a cursor), this project treats the detected frequency as an intermediate representation that can be transformed into color information.

The encoded color information may later be combined with source localization techniques such as sLORETA and targeted brain stimulation using Transcranial Alternating Current Stimulation (tACS) for future investigation of cortical color restoration.

It is important to note that this project does not claim to restore human color perception. Instead, it develops the computational framework required for future research in this area.

# 14. Encoding Framework Philosophy

## Why Do We Need an Encoding Framework?

One of the primary goals of this research is to investigate how visual information can be represented computationally before it is potentially delivered back to the brain through future cortical stimulation techniques.

At first glance, it may appear that this project attempts to directly associate EEG frequencies with the physical frequencies of visible light. However, this is **not** the objective of this research.

Instead, this project proposes a **computational encoding framework** that converts detected SSVEP frequencies into predefined color representations.

---

## Understanding the Difference

There are two completely different types of frequencies involved in this project.

### 1. Frequency of Visible Light

Visible light consists of electromagnetic waves with frequencies ranging approximately from **400 THz to 790 THz**.

These frequencies determine the actual color of light that reaches the human eye.

For example:

| Color | Approximate Frequency |
|--------|----------------------|
| Red | 400 – 484 THz |
| Orange | 484 – 508 THz |
| Yellow | 508 – 526 THz |
| Green | 526 – 606 THz |
| Blue | 606 – 668 THz |
| Violet | 668 – 790 THz |

These frequencies belong to **light itself** and are many trillions of cycles per second.

---

### 2. Frequency of SSVEP Signals

SSVEP frequencies are completely different.

During an SSVEP experiment, a visual target continuously flickers at a specific frequency.

For example,

- 8 Hz
- 10 Hz
- 12 Hz
- 15 Hz

When a participant focuses on one of these flickering targets, neurons in the visual cortex begin responding at approximately the same frequency.

These neural oscillations are recorded using EEG and are known as **Steady-State Visual Evoked Potentials (SSVEPs).**

These frequencies exist only as brain responses and have **no direct physical relationship** with the frequencies of visible light.

---

## Why Can't We Directly Match Them?

A common misconception is to assume that

```
8 Hz = Red
```

or

```
12 Hz = Blue
```

This is scientifically incorrect.

The human brain does **not** naturally interpret an 8 Hz neural oscillation as the color red.

Instead, SSVEP frequencies simply indicate **which visual stimulus the participant is attending to**.

Therefore, an additional computational representation is required.

---

## Our Proposed Encoding Philosophy

This project introduces the idea of using SSVEP frequencies as **control signals** rather than as representations of actual light frequencies.

Instead of saying

```
8 Hz is Red
```

the proposed framework defines

```
Detected SSVEP Frequency
          │
          ▼
Encoding Algorithm
          │
          ▼
Color Representation
```

In other words,

the detected EEG frequency becomes the **input** to an encoding algorithm, which then generates a predefined color representation.

The mapping is **designed by the system**, not by nature.

---

## Analogy with Computers

Modern computers do not store the word "Red."

Instead,

they represent colors using numerical values.

For example,

```
Red

↓

RGB

↓

(255, 0, 0)
```

Similarly,

this project proposes

```
Detected EEG Frequency

↓

Encoding Algorithm

↓

Internal Color Representation
```

The encoding algorithm acts as a translator between decoded brain signals and computational color information.

---

## Why Is This Important?

Separating the detection stage from the encoding stage provides several advantages.

- The EEG decoding pipeline remains independent of the color representation.
- Different encoding strategies can be investigated without changing the signal processing algorithms.
- The framework becomes scalable for future research involving larger color spaces.
- The encoded information may eventually be used by future cortical stimulation systems.

---

## Research Contribution

The primary scientific contribution of this project is **not** the restoration of color perception.

Instead, the contribution is the development of a computational framework that:

- Detects SSVEP frequencies from EEG.
- Converts the detected frequencies into structured color representations.
- Provides an intermediate encoding layer for future source localization and cortical stimulation studies.

This encoding framework serves as a bridge between EEG signal decoding and future brain stimulation research.

---

## Summary

This project **does not claim** that SSVEP frequencies represent the physical frequencies of visible colors.

Instead, it proposes a computational encoding framework in which detected SSVEP frequencies are translated into predefined color representations that may serve as an intermediate layer for future cortical stimulation systems.

The focus of this research is therefore on **encoding**, **signal processing**, and **computational representation**, rather than direct restoration of human color vision.
